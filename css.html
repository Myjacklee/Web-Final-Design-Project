什么是 CSS?

通俗来讲，HTML是对网页的界面进行结构布局的，但是光有布局还远远不够，还需要对布局进行美化，CSS就可以达到对网页界面美化的效果，让界面更加地美观

  • CSS指层叠样式表（Cascading Style Sheets）
  • 样式定义如何显示HTML元素
  • 样式通常储存在样式表中
  • 把样式添加到HTML 4.0中，是为了解决内容与表现分离的问题
  • 外部样式表可以极大提高工作效率
  • 外部样式表通常储存在CSS文件中
  • 多个样式定义可层叠为一个

样式解决了一个很大的问题

HTML标签原本被设计为用于定义文档内容，如下面的实例

  1. <h1>标题一</h1>
  2. <p>段落<p>

样式表定义如何显示 HTML 元素，就像 HTML 中的字体标签和颜色属性所起的作用那样。样式通常保存在外部的 .css 文件中。我们只需要编辑一个简单的 CSS 文档就可以改变所有页面的布局和外观。
